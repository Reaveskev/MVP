# Fitness Tracker

Fitness Tracker can be used to store users and their workouts in a database. The user can then create, read, update, and delete their user information and workouts. They will also be able to see other user workouts.

### What I learned

For this being my first complete fullstack application I am proud of how it functions and looks overall. There is still a lot of room for improvement. I am excited to keep learning and refining my skills.

## Built with

- Semantic HTML5 markup
- CSS3
- JavaScript
- Express
- PostgreSQL
